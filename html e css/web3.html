∇ Definition: What is Web 3.0?

Web 3.0 is the third age of the Internet, where websites and applications will be able to process the human information processing capacity in a form of intelligence, through technologies such as machine learning accounting (Big Data, accounting technology). decentralized (DLT), etc. Web 3.0 was originally called the Semantic Web by the inventor of the World Wide Web Tim Berners-Lee, it was intended to be a smarter and more open Internet. The definition of Web 3.0 can be expanded as follows: data will be interconnected in a decentralized way, which would be a leap forward for the current generation of internet (Web 2.0), where data is mainly stored in our centralized reserves. In addition, users and machines can interact with the data. But for that to happen, programs need to understand how conceptually contextually. With that in mind, the two pillars of Web 3.0 are the semantic web and artificial intelligence (AI).
